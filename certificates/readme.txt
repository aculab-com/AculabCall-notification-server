place your VOIP.pem certificate in this folder

The VOIP.pem file is expected to have the following format:

Bag Attributes
    friendlyName: VoIP Services: ...
    localKeyID: ... 
-----BEGIN CERTIFICATE-----
...
-----END CERTIFICATE-----
Bag Attributes
    friendlyName: ...
    localKeyID: ... 
Key Attributes: <No Attributes>
-----BEGIN PRIVATE KEY-----
...
-----END PRIVATE KEY-----
